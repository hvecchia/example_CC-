# example_CC

Projeto de exemplo criado durante o curso **Claude Code** na Udemy.

## Sobre

Este repositório contém exemplos práticos desenvolvidos ao longo do curso, demonstrando o uso do **Claude Code** — a CLI oficial da Anthropic para interagir com o modelo Claude diretamente no terminal e no ambiente de desenvolvimento.

## O que é o Claude Code?

O [Claude Code](https://claude.ai/code) é uma ferramenta de linha de comando que permite:

- Editar e refatorar código com inteligência artificial
- Executar tarefas de engenharia de software de forma interativa
- Integrar o Claude ao fluxo de trabalho via terminal, VS Code e JetBrains
- Automatizar tarefas repetitivas com hooks e agentes personalizados

## Tecnologias

- Claude Code CLI
- Git / GitHub

## Como usar

1. Instale o Claude Code:
   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

2. Autentique-se:
   ```bash
   claude
   ```

3. Explore os exemplos deste repositório dentro do Claude Code.
